# Example Outputs

This directory contains a small selection of JSON files generated by the scripts located in `edilia/scripts/`.

These examples are provided to illustrate:

* the structure of the output files
* the type of results produced by different scripts
* variability across sites, configurations, and random seeds

---

## Files

### `lasso_PLYMOUTH_90features.json`

Generated using the script:

```bash
python scripts/lasso_test.py --site PLYMOUTH --num_features 90
```

This file contains the results of a LASSO-based feature selection experiment for the PLYMOUTH site using 90 predictors.

---

### `GENERIC_CONFIG/seed_2/strasbourg__npreds90.json`

Generated using the script:

```bash
python scripts/npreds.py --site STRASBOURG --seeds [2] --num_predictors 90
```

This file corresponds to a single run (seed = 2) of the predictor selection procedure for the STRASBOURG site.

---

### `GENERIC_CONFIG/EDINBURGH_npreds90_summary.json`

Generated using the script:

```bash
python scripts/npreds.py --site EDINBURGH --seeds [1,2,3,4,5] --num_predictors 90
```

This file contains aggregated results (averages) computed from five runs corresponding to different random seeds.

It summarizes the outputs of the individual files generated for each seed.

---

## Notes

* These files are provided for demonstration purposes only.
* All results presented in the associated study can be reproduced using the full dataset and the `run.py` pipeline.
* For a complete description of the output format, refer to the documentation in `documentation/outputs.md`.
